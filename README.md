# Excel & Worksheet Skill

## What This Skill Does

| Workflow | Calls | What It Achieves |
|----------|-------|------------------|
| Create Report | 5 | Data → table → chart → save |
| Read & Analyze | 3 | Open → read → describe |
| Build Dashboard | 6 | Data + pivot + chart + formatting |
| Format & Style | 2-3 | Professional appearance |
| Protect & Share | 2 | Lock + save encrypted |

### Chart Types (12)
Column, Bar, Line, Pie, Scatter, Area, Doughnut, Waterfall, Funnel, Treemap, Sunburst, Histogram, Box & Whisker, Map

## Installation

```bash
git clone https://github.com/zavora-ai/skill-excel-worksheet.git \
  ~/.skills/skills/excel-worksheet
```
